# IS211 Assignment 11 – Flask To-Do List App

This project is a simple Flask-based **To-Do List Web Application**.  
Users can add, view, and clear tasks directly through a web interface.

## Features
- Add a task with description, email, and priority (Low, Medium, High)  
- View all added tasks in a table  
- Clear the entire list with one click  
- Basic form validation for required fields and valid email format  

## How to Run
1. Install Flask (if not installed):
   ```bash
   pip install flask
   ```
2. Run the app:
   ```bash
   python todoapp.py
   ```
3. Open in your browser:
   ```
   http://127.0.0.1:5000
   ```

## Files
- `todoapp.py` – Main Flask application  
- `templates/index.html` – HTML template for the web interface
